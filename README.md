# Sinisa Stojanovic Portfolio

This is a personal portfolio web application built with Next.js, React, and Tailwind CSS. It showcases my projects, skills, resume, and contact information.

## Features
- **Project Cards**: Display screenshots, project name, technologies, and links to GitHub and live app. Cards flip on hover and link to detailed project pages.
- **Responsive Design**: Mobile-friendly layout using Tailwind CSS.
- **Contact Form**: Send messages directly from the site.
- **Resume Download**: Preview and download CV as PDF.

## Technologies Used
- Next.js
- React
- Tailwind CSS
- TypeScript
- Shadcn UI
- React Icons

## Getting Started
1. **Install dependencies**:
   ```bash
   npm install
   ```
2. **Run the development server**:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure
- `app/` – Main pages (Home, About Me, Contact, Resume, Work)
- `components/` – Reusable UI components (ProjectCard, Tabs, etc.)
- `data/` – Static data (projects, about me)
- `public/` – Images and resume PDF
- `schemas/` – Validation schemas
- `actions/` – Server actions (contact form)
